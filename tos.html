2. Description du service

Keplerr Bot est un bot Discord développé avec Discord.js qui fournit diverses fonctionnalités pour les serveurs Discord, notamment :

  • Commandes de modération
  • Fonctionnalités de divertissement
  • Outils utilitaires
  • Générateurs et outils spécialisés
  • Configuration personnalisée
  • Intégrations gaming (Minecraft, World of Warcraft)

3. Utilisation acceptable

En utilisant Keplerr Bot, vous vous engagez à :

  • Respecter les Conditions d'utilisation de Discord
  • Ne pas utiliser le Bot pour des activités illégales ou malveillantes
  • Ne pas tenter de contourner les fonctionnalités de sécurité du Bot
  • Ne pas spammer ou abuser des commandes du Bot
  • Respecter les autres utilisateurs et maintenir un environnement positif

4. Permissions et accès

Keplerr Bot nécessite certaines permissions Discord pour fonctionner correctement :

  • Lire les messages : Pour traiter les commandes
  • Envoyer des messages : Pour répondre aux commandes
  • Gérer les messages : Pour les fonctionnalités de modération
  • Utiliser les slash commands : Pour l'interface de commandes moderne
  • Connecter et parler : Pour les fonctionnalités vocales (si applicable)

5. Collecte et utilisation des données

Keplerr Bot peut collecter et traiter certaines données pour fournir ses services :

  • Données de serveur : ID du serveur, nom, paramètres de configuration
  • Données d'utilisateur : ID utilisateur, nom d'utilisateur pour les commandes
  • Contenu des messages : Uniquement pour traiter les commandes
  • Logs d'utilisation : Pour le débogage et l'amélioration du service

Ces données sont utilisées uniquement pour le fonctionnement du Bot et ne sont pas partagées avec des tiers.
